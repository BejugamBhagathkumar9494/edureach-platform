# 🎓 EduReach – AI College Chatbot

EduReach is an AI-powered college assistant that helps students get instant answers about admissions, courses, fee structures, scholarships, placements, and campus facilities through a conversational interface.

## 🚀 Live Demo

🌐 https://edureach-platform-kzjp.vercel.app

## ✨ Features

- 🤖 AI-powered chatbot using Google Gemini
- 🔐 JWT Authentication
- 💬 Real-time chat interface
- 📚 College information assistance
- 📱 Responsive design
- ☁️ Cloud deployment

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **AI:** Google Gemini API
- **Deployment:** Vercel + Render

## ⚙️ Run Locally

```bash
# Clone the repository
git clone https://github.com/your-username/edureach.git

# Install frontend
cd frontend
npm install
npm run dev

# Install backend
cd ../backend
npm install
npm run dev
```
